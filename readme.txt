Nyissuk meg az index.html-t.

Kattintsunk at "Új gyűjtemény létrehozása" gombra.
Ezt követően töltsük ki az összes mezőt.
Megjelenik az első gyűjteményünk, amit át tudunk nevezni, valamint hozzá fel tudunk venni elemeket.
Átnevezés esetén a gomb megnyomását követően írjuk be az új nevet.
Elem hozzáadása esetén szintén írjunk be egy nevet.
	- A beírt névvel megjelenik az első elemünk a gyűjteményben. A gyűjteményhez adhatunk további elemeket is.
	- Minden elem rendelkezik egy ikonnal, amire kattintva három opció közül választhatunk:
		-- Átnevezés esetén egy új név megadását követően frissül majd az elem.
		-- Áthelyezés esetén ha létezik az általunk megadott gyűjtemény, akkor áthelyeződik oda az elem.
		-- Törlés esetén eltűnik az elem.
		

Egyéb információk:
- Visual Studio Code fejlesztői környezetben készült
- A CSS általam lett megvalósítva saját elképzelés alapján.
